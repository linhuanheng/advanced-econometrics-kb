# Ch2: 设定与数据问题 (Specification & Data Problems)

与异方差（相对次要的问题，不影响OLS一致性）不同，本章讨论的问题可导致OLS估计产生**偏误和不一致**。

---

## 1. 函数形式误设 (Functional Form Misspecification)

### 1.1 问题描述

当模型未正确刻画因变量与解释变量之间的关系时，发生函数形式误设。

**典型例子**: 遗漏交乘项

真实模型: $\log(\text{wage}) = \beta_0 + \beta_1 \text{edu} + \beta_2 \text{exper} + \beta_3 \text{exper}^2 + \beta_4 \text{female} + \beta_5 \text{female} \cdot \text{edu} + u$

误设模型: $\log(\text{wage}) = \beta_0 + \beta_1 \text{edu} + \beta_2 \text{exper} + \beta_3 \text{exper}^2 + \beta_4 \text{female} + u$

### 1.2 导致的偏误

| 偏误类型 | 机制 |
|---------|------|
| **所有系数有偏不一致** | 遗漏的 $\text{female} \cdot \text{edu}$ 进入误差项后与 $\text{edu}$ 和 $\text{female}$ 都相关 → 所有参数估计受污染 |
| 偏误方向不确定 | 取决于遗漏项与包含变量间的相关结构 |
| 偏误不随样本增大而消失 | 是系统性的模型设定错误 |

**本质原因**: 函数形式误设是遗漏变量问题的特例——遗漏的变量（如交乘项、平方项）恰好是已包含变量的函数 → 内生性。

### 1.3 解决方案

#### 方法一：RESET检验 (Ramsey, 1969)

**步骤**:
1. 估计原模型 $y = \beta_0 + \beta_1 x_1 + \cdots + \beta_k x_k + u$，得拟合值 $\hat{y}$
2. 扩展模型: $y = \beta_0 + \beta_1 x_1 + \cdots + \beta_k x_k + \delta_1 \hat{y}^2 + \delta_2 \hat{y}^3 + \text{error}$
3. $H_0: \delta_1 = \delta_2 = 0$，用 $F_{2, n-k-3}$ 检验

**检验原理**: $\hat{y}^2$ 和 $\hat{y}^3$ 包含原始变量（$x_j$）的平方项和交叉项信息。若这些项显著 → 原模型遗漏了重要的非线性成分。

**RESET的局限性**:
| 局限 | 含义 |
|------|------|
| 拒绝时不指示方向 | 不知道应该加平方项还是对数变换还是交乘项 |
| 对遗漏变量无检验力 | 若遗漏变量不是 $x_j$ 的函数，RESET不敏感 |
| 不是异方差检验 | 同方差下RESET也可能拒绝（因为非线性） |

#### 方法二：非嵌套模型检验

当面临两个竞争模型（如水平形式 vs 对数形式）时：

**Davidson-MacKinnon检验**:
1. 估计模型B，得拟合值 $\hat{\hat{y}}$
2. 回归 $y = \beta_0 + \beta_1 x_1 + \cdots + \mu \hat{\hat{y}} + \text{error}$
3. $H_0: \mu = 0$ 的t检验显著 → 拒绝模型A

**局限**: 两个模型可能都被拒绝，或都不被拒绝。后者可借助调整 $R^2$ 辅助选择。

#### 方法三：添加灵活的函数形式

- 对关键变量添加二次项
- 使用对数变换（常能减轻异方差和极端值影响）
- 引入交乘项捕捉交互效应

**警告**: 过多变量消耗自由度 → 估计精度下降。

## 2. 代理变量 (Proxy Variables)

### 2.1 问题描述

真实模型含不可观测变量 $x_3^*$（如"能力"）：

$$y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \beta_3 x_3^* + u$$

### 2.2 遗漏变量导致的偏误

若直接忽略 $x_3^*$ 进行OLS：$\text{plim}(\hat{\beta}_1) = \beta_1 + \beta_3 \cdot \frac{\text{Cov}(x_1, x_3^*)}{\text{Var}(x_1)} \neq \beta_1$

- 若 $x_3^*$（能力）与 $x_1$（教育）正相关且 $\beta_3 > 0$ → $\hat{\beta}_1$ **高估**教育的真实回报
- 偏误方向取决于遗漏变量与被包含变量的相关符号

### 2.3 解决方案：代理变量法

用可观测的代理变量 $x_3$（如IQ）替代 $x_3^*$：

$$x_3^* = \delta_0 + \delta_3 x_3 + v_3$$

**"插入法" (plug-in solution)**: 直接用 $x_3$ 替代 $x_3^*$ 进行OLS回归。

#### 一致性条件

| 条件 | 数学表述 | 含义 |
|------|---------|------|
| 误差与所有变量无关 | $u$ 与 $x_1, x_2, x_3, x_3^*$ 不相关 | 代理变量不是内生变量 |
| 代理误差外生 | $v_3$ 与 $x_1, x_2, x_3$ 不相关 | 代理变量的测量误差不系统依赖于其他变量 |
| 冗余条件 | $E(x_3^* \mid x_1, x_2, x_3) = E(x_3^* \mid x_3) = \delta_0 + \delta_3 x_3$ | 一旦控制 $x_3$，$x_1$ 和 $x_2$ 不再影响 $x_3^*$ 的期望 |

**冗余条件**是最关键的——其含义是代理变量 $x_3$ 包含了 $x_3^*$ 的全部相关信息，$x_1$ 和 $x_2$ 不会提供关于 $x_3^*$ 的额外信息。以IQ为能力代理为例：该条件要求给定IQ后，教育年限和经验的差异不再反映能力差异。

#### 条件不满足时的后果

若冗余条件不成立（$x_3^*$ 与 $x_1, x_2$ 在给定 $x_3$ 后仍相关），则代理变量法**不能消除偏误**——$\hat{\beta}_1$ 和 $\hat{\beta}_2$ 仍然有偏。

#### 以IQ为能力代理的实例

| 模型 | $\hat{\beta}_{\text{edu}}$ | 解释 |
|------|--------------------------|------|
| 不含IQ | 0.065 | 可能高估（遗漏能力） |
| 含IQ | 0.054 | 控制能力后教育回报下降 |

IQ每增加一个标准差（15分），收入约增加 $0.0036 \times 15 = 5.4\%$。

### 2.4 替代方案：滞后因变量作为代理

$$\text{crime}_t = \beta_0 + \beta_1 \text{unem}_t + \beta_2 \text{expend}_t + \beta_3 \text{crime}_{t-1} + u$$

**作用**: $\text{crime}_{t-1}$ 间接控制影响犯罪率的历史因素（城市特征、文化等），这些因素通常不可观测。提供了一种不使用正式代理变量而控制不可观测异质性的方法。

## 3. 测量误差 (Measurement Error)

### 3.1 因变量中的测量误差

$$y^* = \beta_0 + \beta_1 x_1 + \cdots + u, \quad y = y^* + e_0$$

| 条件 | 后果 |
|------|------|
| $E(e_0) = 0$, $e_0$ 与所有 $x_j$ 独立 | OLS**无偏一致** |
| $\text{Cov}(e_0, u) = 0$ | $\text{Var}(u+e_0) = \sigma_u^2 + \sigma_0^2 > \sigma_u^2$ |

**偏误分析**: 点估计不受影响，但**误差方差增大** → $R^2$ 降低、标准误变大 → t统计量变小、检验力下降。

⚠ 若 $e_0$ 与 $x_j$ 相关（如高收入者系统性少报储蓄），则OLS有偏——转化为内生性问题。

### 3.2 自变量中的测量误差（CEV假设）

$$y = \beta_0 + \beta_1 x_1^* + u, \quad x_1 = x_1^* + e_1$$

#### CEV假定

| 假定 | 条件 | 后果 |
|------|------|------|
| $\text{Cov}(x_1^*, e_1) = 0$ | $e_1$ 与真实值 $x_1^*$ 不相关 | OLS**不一致**（衰减偏误） |

#### CEV下的衰减偏误

$$\text{plim}(\hat{\beta}_1) = \beta_1 \cdot \frac{\sigma^2_{x_1^*}}{\sigma^2_{x_1^*} + \sigma^2_{e_1}} < |\beta_1|$$

| 符号 | 含义 |
|------|------|
| $\sigma_{x_1^*}^2$ | $x_1^*$ 的真实变异（信号） |
| $\sigma_{e_1}^2$ | 测量误差的变异（噪声） |
| $\frac{\sigma_{x_1^*}^2}{\sigma_{x_1^*} + \sigma_{e_1}^2}$ | 信噪比（总小于1） |

**偏误方向**: 始终向零衰减（attenuation bias）。估计值比真实值更接近零。

**偏误程度**: 取决于信噪比。若 $x_1^*$ 变异大而测量误差小 → 偏误轻微；反之偏误严重。

#### 多元回归中的CEV后果更复杂

$$y = \beta_0 + \beta_1 x_1^* + \beta_2 x_2 + \beta_3 x_3 + u, \quad x_1 = x_1^* + e_1$$

- $\hat{\beta}_1$ 仍存在衰减偏误：$\text{plim}(\hat{\beta}_1) = \beta_1 \cdot \frac{\sigma_{r_1^*}^2}{\sigma_{r_1^*}^2 + \sigma_{e_1}^2}$，$r_1^*$ 是 $x_1^*$ 对其他 $x$ 的回归残差
- **其他系数的估计量 $\hat{\beta}_2, \hat{\beta}_3$ 也一般有偏**（不再仅限于 $\hat{\beta}_1$）
- 偏误方向和大小取决于变量间的相关结构，难以一般化

#### CEV假定不成立的情况

**吸烟与GPA的例子**:
- $x_1^*$（实际吸烟量）和 $e_1$（报告误差）通常相关
- 不吸烟者（$x_1^*=0$）很可能报告 $x_1=0$（$e_1=0$）
- 大量吸烟者可能系统性少报 → $\text{Cov}(x_1^*, e_1) \neq 0$

→ 偏误方向不再可以简单预测。

### 3.3 测量误差的解决方案

| 方法 | 适用场景 | 说明 |
|------|---------|------|
| **IV/2SLS** | 有另一个独立的 $x_1^*$ 度量或有效工具变量 | 最常用的修正方法（详见Ch3） |
| 重复测量 | 有两次对 $x_1^*$ 的独立测量 | 用一次测量作为另一次的工具变量 |
| 使用代理变量 | 有 $x_1^*$ 的代理变量 | 如第2节所述 |
| **对数变换**（对因变量） | 因变量中存在乘性测量误差 | $\log(y) = \log(y^*) + e_0$，$e_0$ 独立于 $x$ 时OLS仍一致 |

## 4. 数据问题

### 4.1 缺失数据

| 缺失机制 | 偏误后果 | 解决方法 |
|---------|---------|---------|
| **完全随机缺失** (MCAR) | **无偏误**，仅样本量减少 → 标准误增大 | 直接使用完整观测 |
| **随机缺失** (MAR) | 缺失与可观测变量相关但与缺失值本身无关 → **无偏误** | 直接使用完整观测 |
| **非随机缺失** (NMAR) | 缺失概率与被缺失变量的值相关 → **有偏不一致** | 需用Heckman型选择模型 |

**随机缺失的经济学例子**: 在出生体重数据中，若低教育水平者更可能缺失教育变量 → 非随机缺失（因为缺失概率与"教育"这个变量的值相关）。

### 4.2 非随机抽样

#### 外生样本选择（基于自变量选择）

**例子**: 基于收入水平分层抽样来估计储蓄函数

$$\text{saving} = \beta_0 + \beta_1 \text{income} + \beta_2 \text{age} + \beta_3 \text{size} + u$$

- 若仅对 $\text{income} < 100,000$ 的人群抽样 → **OLS无偏**（只要高收入人群内的关系与全人群一致）
- 前提: $E(\text{saving} \mid \text{income}, \text{age}, \text{size})$ 在子人群中不变

#### 内生样本选择（基于因变量选择）

**例子**: 仅对财富 $< 250,000$ 的人估计财富方程

$$\text{wealth} = \beta_0 + \beta_1 \text{edu} + \beta_2 \text{exper} + \beta_3 \text{age} + u$$

→ **OLS有偏不一致**。因为截断后的条件期望不等于总体的线性回归函数。

#### 分层抽样

| 分层类型 | 例子 | 偏误 |
|---------|------|------|
| 外生分层 | 过度抽样女性以研究军队薪酬 | OLS**无偏一致** |
| 内生分层 | 过度抽样低薪军人以研究军队薪酬 | OLS**不一致** |

### 4.3 离群值 (Outliers)

#### 为什么OLS对离群值敏感

OLS最小化**残差平方和**: $\min \sum_{i=1}^n (y_i - X_i b)^2$

- 一个残差为 $10$ 的观测对目标函数的贡献是残差为 $1$ 的观测的 $100$ 倍
- 离群值对OLS估计施加不成比例的"杠杆"影响 → 估计值可能发生剧烈变化

**例子（R&D强度与公司规模）**:
- 32家化学公司: $\widehat{\text{rdintens}} = 2.625 + 0.000053 \text{Sales} + 0.0446 \text{profmarg}$（Sales不显著）
- 剔除1家异常公司后: $\widehat{\text{rdintens}} = 2.297 + 0.000186 \text{Sales} + 0.0478 \text{profmarg}$（Sales系数变为3倍多，t统计量超过2）

#### 解决方案

**方法一：LAD (Least Absolute Deviations)**

$$\min_{b_0,...,b_k} \sum_{i=1}^n |y_i - b_0 - b_1 x_{i1} - \cdots - b_k x_{ik}|$$

| LAD vs OLS | 说明 |
|------------|------|
| 目标函数 | 残差绝对值之和（vs OLS的残差平方和） |
| 估计对象 | **条件中位数**（vs OLS的条件均值） |
| 离群值韧性 | 强——因为残差以线性而非平方方式进入目标函数 |
| 有限样本推断 | 无精确分布，仅渐近有效 |
| 计算 | 无闭式解，需迭代算法（大样本时效率低） |

**LAD注意**: 若误差分布不对称，LAD（中位数）和OLS（均值）给出不同的偏效应估计——这可能是分布特征而非离群值问题。

**方法二：其他措施**

1. 检查数据的最大值/最小值 → 发现录入错误
2. 对数变换 → 显著降低极端值的杠杆
3. 报告含离群值和不含离群值的OLS结果
4. 使用稳健回归（Huber, Tukey bisquare）

## 5. 问题汇总：偏误诊断与解决路径

| 问题 | 是否导致OLS偏误/不一致 | 诊断工具 | 解决方案 |
|------|----------------------|---------|---------|
| 函数形式误设 | **是**（所有系数） | RESET, 非嵌套检验 | 添加平方项/交乘/对数 |
| 遗漏变量（无代理） | **是** | — | 代理变量 / IV / 面板数据 |
| 代理变量不当 | **是**（无法消除偏误） | 理论论证冗余条件 | 寻找更优代理 / IV |
| 因变量测量误差 | **否**（若$e_0$外生），仅方差变大 | — | 报告OLS即可 |
| 自变量测量误差(CEV) | **是**（衰减偏误） | IV检验 | IV/2SLS |
| 缺失数据(随机) | **否** | — | 直接忽略缺失观测 |
| 缺失数据(非随机) | **是** | — | 选择模型 |
| 外生样本选择 | **否** | — | OLS |
| 内生样本选择 | **是** | — | Heckman两步法 |
| 离群值 | **是**（有限样本严重） | 残差图, DFBETA | LAD / 对数变换 / 稳健回归 |
