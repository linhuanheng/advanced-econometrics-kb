# 高级计量经济学知识库 (Advanced Econometrics Knowledge Base)

基于暨南大学 IESR 研究生课程《高级计量经济学 III》（Graduate Econometrics III, Spring 2025）讲义整理的结构化知识库。

## 内容概览

| 章节 | 主题 | 核心内容 |
|------|------|---------|
| Ch1 | 异方差 (Heteroskedasticity) | White稳健标准误、BP/White检验、WLS/FGLS |
| Ch2 | 设定与数据问题 (Specification & Data Problems) | 函数形式误设、代理变量、测量误差、离群值 |
| Ch3 | 工具变量估计 (IV & 2SLS) | IV条件、识别、2SLS渐近理论、弱工具变量 |
| Ch4 | 单方程扩展专题 | 控制函数、设定检验、CRC模型、双重差分(DID) |
| Ch5 | 线性非观测效应面板数据模型 | RE/FE/FD估计、Hausman检验、严格外生性 |
| Ch6 | 高级面板数据专题 | 序贯外生性、动态面板、Chamberlain方法、Hausman-Taylor |
| Ch7 | 二元响应模型 | LPM、Logit/Probit、MLE、面板二元响应模型 |
| Ch8 | 处理效应评估 | ATE/ATT/LATE、可忽略性、倾向得分、IV、断点回归 |

## 教材

- Wooldridge, J. M., *Introductory Econometrics: A Modern Approach*
- Wooldridge, J., *Econometric Analysis of Cross-Section and Panel Data*, MIT Press, 2010
- Cameron, A. C. & Trivedi, P. K., *Microeconometrics: Methods and Applications*
